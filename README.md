Yoga Website

Kani waa website fudud oo loogu talagalay adeegyada Yoga iyo Caafimaadka. Waxaa lagu dhisay HTML iyo CSS, isagoo diiradda saaraya quruxda iyo isticmaalka fudud.

## Muuqaalka Mashruuca
* **Responsive Design:** Wuxuu si habsmooth ah ugu shaqeeyaa Mobile iyo Desktop.
* **Clean UI:** Naqshad dagan oo ku habboon mawduuca Yoga.
* **Sections:** Waxay ka kooban tahay Home, About, Classes, iyo Contact.
